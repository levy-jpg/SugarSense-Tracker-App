# App Name: SugarSense

SugarSense is a user-friendly cross-platform mobile application designed to help individuals track and manage their blood glucose levels. It empowers users to make informed health decisions by offering a simple, intuitive interface for monitoring blood sugar trends, setting goals, and staying within target ranges. Tailored to those managing diabetes or aiming for healthy blood sugar levels, SugarSense provides essential tools for health management.

# Features:
•	Add Sugar Level:
Easily logs blood sugar readings with the date and time for efficient tracking.

•	Sugar Level History:
View a real-time graph of your blood sugar levels over time, making trends and progress easy to track.

•	Average Sugar Level Display:
Automatically calculate and display the average blood sugar level, helping users assess whether they're within their target range.

•	Data Persistence:
Store data locally on the device, ensuring easy access even when the app is closed or the device is restarted.

•	Dark/Light Mode:
Customize the app's appearance with a toggle between dark and light mode, based on user preference.

•	Clear All Data:
Take control of your data with an option to clear all information stored in the app.
SugarSense combines functionality with clean, intuitive design, offering seamless navigation and effective data visualizations to help users stay on top of their health and make informed decisions with confidence.
